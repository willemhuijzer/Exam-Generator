Question 1:
What is the main motivation for exploring parameter control in Evolutionary Algorithms?
A. To complicate the optimization process
B. To set fixed parameter values for all stages of the search
C. To find optimal parameter values that may vary throughout the search
D. To limit the influence of the evolutionary process on parameter values

Answer: C

Question 2:
Which of the following is NOT a parameter that can be controlled using parameter control techniques in Evolutionary Algorithms?

D. Population size

Answer: C
