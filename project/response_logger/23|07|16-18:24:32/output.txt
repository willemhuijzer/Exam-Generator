Question 1:
What is the motivation behind parameter control in Evolutionary Algorithms?
A. To find optimal parameter values
B. To increase user control over the algorithm
C. To reduce the computing resources required
D. To improve the predictability of parameter values

Answer: A

Question 2:
Which of the following is not a strategy parameter in Evolutionary Algorithms?
A. Mutation operator and mutation rate
B. Crossover operator and crossover rate
C. Selection mechanism and selective pressure
D. Evaluation function and fitness value

Answer: D

Question 3:
Which option for varying mutation step size allows for the most user control of the parameter?
A. Option 1
B. Option 2
C. Option 3
D. Option 4

Answer: A

Question 4:
What is the primary feature that distinguishes various forms of parameter control in Evolutionary Algorithms?
A. Evidence/data backing up the changes
B. What component of the algorithm is changed
C. Level/scope of the parameter change
D. Progress made in the search process

Answer: B

Question 5:
Which type of evidence for parameter change is based on comparing values through solutions created with different parameter values?

Question 7:
Which level/scope of parameter change affects the fitness function in Evolutionary Algorithms?

Question 8:
Which type of parameter control relies on predefined event triggers for parameter changes?

Question 9:
What is the main advantage of self-adaptive parameter control in Evolutionary Algorithms?
A. Increased user control
B. Reduction in required computing resources
